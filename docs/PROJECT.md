# AirMaven — 실시간 공개정보(OSINT) 융합 상황인식 글로브

> 대안 명칭: `AirMaven: 다출처 실시간 위협 상황판` · `D4D AirMaven`
> 라이브 데모: **https://d4d.n2f.site**

---

## 한 줄 소개

공개 항공(ADS-B)·해상(AIS)·위성(TLE)·OSINT(텔레그램) 데이터를 하나의 실시간 3D 지구본에 융합하고, 텔레그램 주장에 **신뢰도 점수**를 매겨 검증하며, 모바일 텔레그램 봇으로 조회·경보까지 제공하는 **방어적 상황인식 플랫폼**.

## 해결하려는 문제

분쟁·공역 상황인식에 필요한 공개정보(OSINT)는 **파편화**돼 있다. 분석가는 ADS-B 사이트, AIS, 텔레그램 채널, 공습경보, 위성 궤도, 지진, NOTAM을 **각각** 열어 대조해야 하고, 텔레그램의 "타격 주장"은 검증 없이 그대로 소비되기 쉽다.

AirMaven은 이 소스들을 **하나의 실시간 지도에 융합**하고, 무엇보다 **주장의 신뢰도를 투명하게 채점**(교차출처 × NASA 열적 × 지역 화재활동)해 미검증 정보가 확정된 것처럼 보이지 않게 한다.

## 핵심 기능

- **실시간 융합 글로브** — 전 세계 항공기·선박·위성을 하나의 3D 지구본에 실시간 표시. 팬/줌 시 뷰포트 기준 필터.
- **위성 실시간 궤도** — Celestrak TLE를 브라우저에서 SGP4 전파(satellite.js), 줌인 시 화면 영역 위성만.
- **OSINT 신뢰도 검증** — 84개 대형 텔레그램 채널 수집 → 전쟁-결과 필터 → 지오로케이트 → **verdict(확인/가능성/미확인/허위) + 0–100% + 근거 원장**. NASA FIRMS 실시간 열적 + 교차출처로 교차검증(웹 패널·봇 동일 로직).
- **위협 레이어** — 우크라이나 공습경보(air_alert_ua), EMSC 얕은 진원(폭발형) 지진, ROK 공역(KADIZ·P-73)·항공로, OpenAIP 공역, FAA NOTAM.
- **AI 코파일럿** — OpenAI 기반 지역 상황 한국어 요약·이상탐지·멀티턴 대화.
- **타임라인 리플레이** — 최근 6시간 항적·선박·위성 이동 재생.
- **텔레그램 봇** — 모바일 조회(`/status`·`/mil`·`/verify`·`/ai`) + 능동 푸시(6시간 분쟁 브리핑, 이상 항적, 공습경보).

## 차별점 — "정직한 검증"

- **날조 없음**: 합성 데이터 미생성. 채널·좌표·열점은 라이브 검증된 실데이터만 사용.
- **투명한 신뢰도**: 근거가 약하면 점수를 억지로 올리지 않는다(교차출처만이면 낮게 유지). 열적 근거는 시각·거리 인과 창을 지켜 "지역에 불남 ≠ 특정 타격 확인"을 구분.
- **비표적화 원칙**: 공개정보 융합·상황인식 보조이며 식별·표적화·교전 판단이 아님을 UI 전반에 명시.
