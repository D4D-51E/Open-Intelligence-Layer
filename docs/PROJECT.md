# AirMaven — 공개정보(OSINT) 실시간 융합·검증 상황인식 플랫폼

> **`AirMaven`은 Project Maven의 의도적 반전이다.** 기밀 센서로 표적을 찾는 대신, **공개정보를 교차검증해 누구나 신뢰할 수 있게** 만든다.
> *Open · Verifiable · Non-targeting · 폰 안의 상황실*
> 라이브 데모: **https://d4d.n2f.site**

---

## 한 줄 소개

공개 항공(ADS-B)·해상(AIS)·위성(TLE)·OSINT(텔레그램) 데이터를 하나의 실시간 3D 지구본에 융합하고, 텔레그램 주장에 **신뢰도를 판정(verdict + 신뢰도% + 근거)**하며, 웹 대시보드와 텔레그램 봇으로 **조회·조기경보(I&W)**까지 제공하는 방어적 상황인식 플랫폼.

## 해결하려는 문제

분쟁·공역 상황인식에 필요한 공개정보는 **파편화**돼 있고, 텔레그램의 "타격 주장"은 **검증 없이 사실처럼 소비**된다. 분석가는 ADS-B·AIS·텔레그램·공습경보·위성·지진·NOTAM을 각각 열어 대조해야 한다. AirMaven은 이 소스들을 **한 화면에 융합**하고, 무엇보다 **집계(aggregation)가 아니라 판정(adjudication)** — 주장이 사실인지 독립 신호로 교차검증한다.

## 어떻게 (핵심 동작)

- **다도메인 실시간 융합 글로브** — 전 세계 항공기·선박·위성을 3D 지구본에, 뷰포트/줌 기준 필터로.
- **OSINT 신뢰도 판정** — 84개 대형 텔레그램 채널 → 전쟁-결과 필터 → 지오로케이트 → **verdict(확인/가능성/미확인/허위) + 0–100% + 근거 원장**. **NASA FIRMS 실시간 열적 × 교차출처**로 교차검증. 근거가 약하면 점수를 억지로 올리지 않음(정직성 엔지니어링).
- **조기경보 레이어** — 우크라이나 공습경보, EMSC 얕은 진원(폭발형) 지진, ROK 공역(KADIZ·P-73)·항공로, OpenAIP 공역, NOTAM.
- **AI 코파일럿** — OpenAI 기반 한국어 지역 요약·이상탐지·멀티턴 대화(제공 데이터에만 근거·출처 표기).
- **텔레그램 봇** — 모바일 조회(`/status`·`/mil`·`/verify`·`/ai`) + 능동 푸시(6h 브리핑 · 이상 항적 · 공습경보).
- **타임라인 리플레이** — 최근 6시간 항적·선박·위성 이동 재생.

## 군 적용 가능성 (핵심 강조)

- **비밀분류 장벽 없음** — 100% 공개정보라 **UNCLASS 레벨 즉시 운용**, 기밀을 못 나누는 **연합/동맹 정보공유**에 특히 강함.
- **결정지원·조기경보(I&W) 역할** — 표적화가 아니므로 법적·ROE 문턱이 낮아 **야전 도입이 빠름**. 비상 스쿼크·고가치 자산·공습경보 푸시가 force protection·항공상황도·ISR 큐잉에 직결.
- **이식성** — 서버리스 + 컨테이너 수집기라 **온프레미스·에어갭·정부클라우드 재배치** 가능(SaaS 락인 없음). 봇 전송계층 독립적 → 실배치 시 승인된 국방 메신저로 어댑터만 교체.
- **ROK 특화** — KADIZ·P-73·ROK 항공로 + 한국어 AI.

## 기술 스택

| 영역 | 기술 |
|---|---|
| 프론트엔드 | Vite · React 18 · TypeScript · MapLibre GL(3D 글로브) |
| 지도/궤도 | OpenAIP 벡터 타일 · satellite.js(브라우저 내 SGP4 전파) |

| 데이터베이스 | Neon Postgres (HTTP 드라이버, 시계열) |
| 상주 수집기 | Railway 워커 — AISStream WebSocket + adsb.lol 폴링 |
| 스케줄링 | GitHub Actions cron (수집 15분 / 브리핑 6시간 / 실시간 경보 5분) |
| AI · 봇 | OpenAI(gpt-4o-mini) · Telegram Bot(웹훅 + 능동 푸시) |
| 데이터 소스 | 공개 19종 (→ [DATA-SOURCES.md](./DATA-SOURCES.md)) |

## 정직한 한계 (설계에 반영)

합성 데이터 미생성(라이브 검증된 실데이터만). 위성은 TLE 계산 위치(실측 아님). 이스라엘 경보는 oref.org.il 해외 IP 차단으로 **동일 공식 데이터의 Tzeva Adom 미러** 사용. 크론 기반이라 "즉시"는 ~5–15분 근사(실배치 시 상주 워커로 상향 가능). 텔레그램은 승인 보안망 아님 → 전송계층 교체 전제.
